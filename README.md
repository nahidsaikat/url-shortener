# url-shortener
URL Shortener Service

## Instructions
* `git checkout https://github.com/nahidsaikat/url-shortener.git`
* `cd url-shortener`
* `python manage.py makemigrations`
* `python manage.py migrate`
* `python manage.py runserver`

### Tests
* `python manage.py test`

### AWS Infrastructure
* `terraform init`
* `terraform plan`
* `terraform apply`
